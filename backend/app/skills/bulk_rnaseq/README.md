# Bulk RNA-seq Skill

Phase 1 skill definition for Bulk RNA-seq count matrix plus metadata workflows.

This skill currently defines contracts only:

- Supported file types.
- Required metadata fields.
- QC metric names.
- Primary and validation method placeholders.
- Output artifacts.
- Reliability rules.

Real DESeq2, edgeR, and limma-voom execution should be connected through the
runner interfaces after the MVP API chain is stable.
